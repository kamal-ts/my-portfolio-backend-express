# Tags API Spec

## Create Tags

Endpoint : POST /api/tags

Request body :

```json
{
    "title": "express js",
    "color_body": "#312321",
    "color_text": "#7347342"
}
```

Response body:

```json
{
    "data": {
        "title": "express js",
        "color_body": "#312321",
        "color_text": "#7347342"
    }
}
```
